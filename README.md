# Intercar Logística

Landing page de práctica realizada por Romina Herrera para mostrar maquetación y diseño web front-end.

Demo en vivo: https://romherre.github.io/intercar-logistica/

## Sobre el proyecto

Sitio de una sola página para una empresa ficticia de logística y e-commerce. Es un proyecto de práctica, pensado para entrenar estructura de landing pages, secciones de servicios, formulario de contacto y diseño responsive.

## Secciones

- Hero: presentación de la marca y llamado a la acción
- Servicios: Logística Flex, Fulfillment y Logística Inversa
- Diferenciales: beneficios destacados de la propuesta
- Cotizar: formulario de contacto
- Footer: datos de contacto y redes (de ejemplo)

## Tecnologías

- HTML5 y CSS3 (archivos separados: index.html, style.css, script.js)
- JavaScript vanilla para la interacción del formulario
- Tipografía Urbanist (Google Fonts) e íconos de Font Awesome
- Diseño responsive con CSS Grid y Flexbox

## Deploy

Publicado con GitHub Pages desde la rama main.

---

Nota: los datos de contacto (teléfono, redes, email) son de ejemplo y no corresponden a una empresa real.
